- [ ] 错误处理（网络断开、API 超时）

- [ ] 记忆管理界面（查看/编辑/删除）

### Step 4：陪伴决策引擎
- [ ] 定时器（每 5 分钟评估）
- [ ] 前置过滤器（全屏/免打扰/冷却/深夜不活跃）
- [ ] 规则 1：早安问候
- [ ] 规则 2：长时间工作提醒
- [ ] 规则 3：深夜提醒
- [ ] 规则 4：空闲超时
- [ ] 规则 5：提醒到期
- [ ] 规则 6：活动切换
- [ ] Level 1 消息生成（固定模板）
- [ ] Level 2 消息生成（模板 + 记忆填充）
- [ ] Level 3 消息生成（LLM 调用）
- [ ] 连接气泡组件展示

### Step 5：Live2D 表情映射
- [ ] idle → 默认表情 + 呼吸
- [ ] speaking → 开心表情 + 说话动作
- [ ] thinking → 歪头（LLM 生成时）
- [ ] sleeping → 困倦表情（深夜）
- [ ] concerned → 担心表情
- [ ] 气泡出现 → 自动切换 speaking
- [ ] 气泡消失 → 恢复 idle

### Step 6：防打扰机制
