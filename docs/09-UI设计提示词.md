# UI 设计提示词 - 桌面陪伴体

> 用这些提示词在 GPT / Midjourney / DALL-E 中生成 UI 参考图。
> 建议用英文生成，效果更好。

---

## 整体风格定义（每次生成都带上这段）

```
I'm designing a desktop AI companion widget for Windows 11.
It's NOT a chat app - it's a small floating character/avatar that lives
on your desktop like a tiny companion creature.

Design language:
- The companion is a small cute avatar/character (80-100px)
- It floats on the desktop, always on top of other windows
- It has speech bubbles that appear when it wants to talk
- Dark theme, purple-blue accent (#6366f1)
- Glassmorphism, soft shadows, rounded everything
- The vibe: cozy, warm, minimal, alive
- Think: desktop pet meets modern AI assistant
- Reference: Clippy but actually useful and beautiful
```

## 图 3：展开式对话面板

```
A Windows 11 desktop. Bottom-right corner shows an expanded chat panel
(about 380x600px) that grew out from the small companion avatar.

The panel has:
- Rounded corners, dark glassmorphism background
- Top: small companion avatar + name "Companion" + close button
- Chat area with messages:
  AI: "Good morning! Any plans for today? ☀️"
  User: "Gonna finish that feature I was working on"
  AI: "The API integration one? You were stuck on the auth part yesterday.
       Want me to help you think through it? 🤔"
- Bottom: rounded input field with send button
- The panel connects visually to the small avatar (it grew from it)
- The desktop/code editor is still visible behind (panel is not fullscreen)

Style: modern chat panel, dark theme, feels like texting a friend,
not like using a chatbot. Cozy and warm.
```
