# AI 桌面陪伴助手 - 项目总览

## 项目一句话描述
一个轻量级 Windows 11 桌面应用，AI 能主动聊天、感知用户状态、记住对话、执行简单任务。

## 项目代号
**Companion**（你可以改成任何你喜欢的名字）

## 你的身份
- 角色：产品经理 + 项目经理（由你担任）
- 开发者：AI（Claude Code / Cursor / 其他 AI 编程工具）
- 你的工作：理解需求 → 写清楚提示词 → 验证 AI 产出 → 反馈修改
